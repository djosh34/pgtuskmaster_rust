# Tutorials

This chapter contains hands-on lessons that guide you through practical activities with pgtuskmaster. Each tutorial takes you step-by-step through a concrete goal, helping you learn by doing.

## What you'll find here

- [First HA Cluster](first-ha-cluster.md) - Start a three-node PostgreSQL HA cluster on your local machine and identify the current leader
- [Single-Node Setup](single-node-setup.md) - Run the smallest shipped deployment: one pgtuskmaster node and one etcd service
- [Observing a Failover Event](observing-failover.md) - Watch a real high-availability failover unfold and observe state changes during primary outages
- [Debug API Usage](debug-api-usage.md) - Read the debug API to inspect cluster state and poll incrementally using `since=` parameters

These tutorials are the best starting point when you want to learn the system by doing. After you are comfortable with the flows here, use the how-to guides for specific operational tasks and the reference pages for exact API and configuration details.
